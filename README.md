# chloejay.github.io
## Personal website 

Build Environment
```bash
$ rbenv install 2.7.0
$ rbenv global 2.7.0

$ rbenv versions
system
  2.6.0
* 2.7.0 (set by RBENV_VERSION environment variable)

$ ruby -v 
2.7.0 

$ gem install jekyll bundler 
$ jekyll -v
4.0.0 
```

Using <a href='https://jekyllrb.com/'>Jekyll</a> and a customised <a href='https://github.com/jekyll/minima'>Minima</a> theme.
